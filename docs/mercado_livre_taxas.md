# Mercado Livre — Taxas e Custos de Venda

_Última atualização: 2026-05-12_
_Fonte: documentação oficial ML para sellers_

## Comissões por Tipo de Anúncio

| Tipo | Comissão | Observações |
|---|---|---|
| Clássico | ~12–16% | Varia por categoria |
| Premium | ~16–19% | Mais exposição, custo maior |

> Confirmar a comissão exata da categoria do produto antes de precificar.

## Frete

| Modalidade | Custo para o Vendedor |
|---|---|
| Mercado Envios Full | Gerenciado pelo ML (estoque no CD deles) |
| Mercado Envios Flex | Entrega no mesmo dia — requer proximidade do comprador |
| Envio normal (Correios/transportadora) | Pago pelo vendedor ou embutido no preço |

## Impostos MEI

| Imposto | Valor |
|---|---|
| DAS mensal (comércio) | ~R$70–75/mês fixo |
| Limite de faturamento anual | R$81.000 |

## Fórmula de Margem Líquida

```
Lucro = Preço_venda - Custo_produto - (Preço_venda × comissão_ML) - Frete - Embalagem

Margem % = Lucro / Preço_venda × 100
```

**Margem mínima aceitável: 25%**

## Exemplo Prático — TC-90

| Item | Valor |
|---|---|
| Custo do produto | R$35,00 |
| Preço de venda | R$65,00 |
| Comissão ML (~16%) | R$10,40 |
| Frete estimado | R$12,00 |
| Embalagem | R$2,00 |
| **Lucro líquido** | **R$5,60** |
| **Margem real** | **~8,6%** |

> Conclusão: TC-90 a R$65 tem margem ruim. Preço ideal seria ~R$72–75 para atingir 25%.

## Notas
- Anúncios pausados por mais de 60 dias podem perder relevância no algoritmo
- Reputação do seller impacta posicionamento — evitar cancelamentos e reclamações
